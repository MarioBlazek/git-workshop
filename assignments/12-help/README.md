Git help
========

Goal
----

Be able to fine more info about Git command and Git itself.

Tasks
-----

* Open man pages for `git config`
* Get more info about `git add` in command line


Hints
-----

* `git help <verb>`
* `man git-<verb>`
* `git <verb> --help`

Solution
--------

* `git help config`
* `git add -h`
